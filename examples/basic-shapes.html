E-Ink Display Simulator - Basic Shapes (296x128)

Usage Example

// Create simulator
const canvas = document.getElementById('display');
const sim = new EinkSimulator.EinkSimulator(canvas, {
  width: 296,
  height: 128
});

// Draw shapes
sim.clear();
sim.fillRect(10, 10, 50, 30);
sim.drawCircle(100, 50, 20);
sim.drawText('Hello World', 150, 50, myFont);

// Export
sim.download('my-display.png');
